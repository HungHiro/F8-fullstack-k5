# Câu lệnh rẽ nhánh
- Thực thi đoạn code dựa vào điều kiện
```scss
@if dieukien {
    // chạy khi điều kiện đúng
} @else {
    // chạy khi điều kiện sai
}
```
Các toàn tử:
```- == : bằng
- >  : lớn hơn
- >= : lớn hơn or bằng
- <  : nhỏ hơn
- <= : nhỏ hơn or bằng
- != : khác
```
Các toán tử kết hợp:
```
and: và
or: hoặc
not: phủ định
```
Các toán tử số học
```
+
-
*
/
% --> chia lấy dư
```
Giá trị đúng sai (Boolean)
```
true --> đúng
false --> sai
```
